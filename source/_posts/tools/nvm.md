##概述
计算机上管理多个nodejs版本的安装

##安装
下载地址: https://github.com/coreybutler/nvm-windows
:::default no-icon
路径名只能为英文,路径名中不能有空格
:::

##命令
nvm list available
查看nodejs所有版本

nvm install <版本号>
安装指定版本nodejs

nvm list 
查看已安装的nodejs版本

nvm use <版本号>
切换到指定版本的nodejs

nvm uninstall <版本号>
卸载指定版本的nodejs
